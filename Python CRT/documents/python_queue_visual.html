📝 Simple Queue Implementation

class Queue: def __init__(self): self.items = [] def enqueue(self, item): # Add item to rear (end) self.items.append(item) def dequeue(self): # Remove and return front item if self.is_empty(): return None return self.items.pop(0) def peek_front(self): # Look at front item without removing if self.is_empty(): return None return self.items[0] def peek_rear(self): # Look at rear item without removing if self.is_empty(): return None return self.items[-1] def is_empty(self): return len(self.items) == 0 def size(self): return len(self.items)

🚀 How to Use

# Create a queue queue = Queue() # Add items (enqueue) queue.enqueue("First") queue.enqueue("Second") queue.enqueue("Third") # Check front item print(queue.peek_front()) # Output: First # Remove items (dequeue) print(queue.dequeue()) # Output: First print(queue.dequeue()) # Output: Second # Check size print(queue.size()) # Output: 1

🆚 Stack vs Queue Comparison

📚 Stack (LIFO)

Last In, First Out

  • Add/Remove from same end (top)
  • Like a stack of plates
  • Operations: push, pop, peek
  • Use: Undo operations, function calls

🚶‍♂️ Queue (FIFO)

First In, First Out

  • Add at rear, remove from front
  • Like a line of people
  • Operations: enqueue, dequeue, peek
  • Use: Task scheduling, printing

💡 Key Concepts

FIFO - First In, First Out

Items are removed in the same order they were added. The first item enqueued is the first to be dequeued.

Main Operations

  • Enqueue: Add item to rear
  • Dequeue: Remove item from front
  • Peek Front: Look at front item
  • Peek Rear: Look at rear item

Common Uses

  • Task scheduling in operating systems
  • Print job management
  • Breadth-first search algorithms
  • Customer service systems
  • Data buffering

⚡ Efficient Queue Implementation

Note: Using collections.deque for better performance

The simple implementation above uses list.pop(0) which is O(n). For better performance, use collections.deque:

from collections import deque class EfficientQueue: def __init__(self): self.items = deque() def enqueue(self, item): # O(1) - Add to right end self.items.append(item) def dequeue(self): # O(1) - Remove from left end if self.is_empty(): return None return self.items.popleft() def peek_front(self): if self.is_empty(): return None return self.items[0] def is_empty(self): return len(self.items) == 0 def size(self): return len(self.items)
