📝 Simple Stack Implementation

class Stack: def __init__(self): self.items = [] def push(self, item): # Add item to top self.items.append(item) def pop(self): # Remove and return top item if self.is_empty(): return None return self.items.pop() def peek(self): # Look at top item without removing if self.is_empty(): return None return self.items[-1] def is_empty(self): return len(self.items) == 0 def size(self): return len(self.items)

🚀 How to Use

# Create a stack stack = Stack() # Add items (push) stack.push("First") stack.push("Second") stack.push("Third") # Check top item print(stack.peek()) # Output: Third # Remove items (pop) print(stack.pop()) # Output: Third print(stack.pop()) # Output: Second # Check size print(stack.size()) # Output: 1

💡 Key Concepts

LIFO - Last In, First Out

Items are removed in reverse order of insertion. The last item pushed is the first to be popped.

Main Operations

  • Push: Add item to top
  • Pop: Remove item from top
  • Peek: Look at top item
  • Is_empty: Check if stack is empty

Common Uses

  • Function calls (call stack)
  • Undo operations
  • Browser back button
  • Expression evaluation
